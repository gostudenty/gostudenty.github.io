13. 3 pieces of paper = 2 ballpens 3 ballpens = 4 pencils 5 pencils = 3 erasers How many pieces of paper can you get for 3 erasers? with linear equation and solution please .. i only know na sagot is 11.25 but i need the solution with a linear equation, thank you!


Let's assign variables to represent the unknown quantities in the problem.

Let x be the number of pieces of paper, y be the number of ballpens, z be the number of pencils, and w be the number of erasers.

From the first statement, we have:

3x = 2y

Solving for y, we get:

y = (3/2)x

From the second statement, we have:

3y = 4z

Solving for z, we get:

z = (3/4)y = (9/8)x

From the third statement, we have:

5z = 3w

Solving for w, we get:

w = (5/3)z = (15/8)x

Now, we can use the third equation to find how many pieces of paper we can get for 3 erasers:

w = (15/8)x

3 = (15/8)x

x = 8/5

So we can get 8/5 pieces of paper for 3 erasers.

Therefore, the solution is:

You can get 8/5 pieces of paper for 3 erasers.

1. Bookmarking - ito ay talaan ng address ng isang web page sa iyong computer upang madali mo itong mahanap muli.

2. Bookmark - ang bookmark ay nagsisilbi lamang bilang isang shortcut pabalik sa website.

3. Cloud - ang cloud ay ang paghahatid ng iba't ibang serbisyo sa pamamagitan ng Internet, kabilang ang pag-iimbak ng data, mga server, database, networking, at software.

4. Web page - ang Web page ay isang dokumento para sa World Wide Web na isang natatanging unipormeng resource locator (URL).

5. Web site - isang hanay ng mga kaugnay na web page na matatagpuan sa ilalim ng iisang domain name, na karaniwang ginagawa ng isang tao o organisasyon.

1. reaction to arturo luz candle vendor ​


Answer:

Arturo R. Luz, painter, sculptor, and designer for more than 40 years, created masterpieces thatexemplify an ideal of sublime austerity in expression and form. From the Carnival series of the late 1950sto the recent Cyclist paintings, Luz produced works that elevated Filipino aesthetic vision to new heightsof sophisticated simplicity. By establishing the Luz Gallery that professionalized the art gallery as aninstitution and set a prestigious influence over generations of Filipino artists, Luz inspired and developeda Filipino artistic community that nurtures impeccable designs.Among his other significant paintings areBagong Taon,Vendador de Flores,Skipping Rope,CandleVendors,Procession,Self-Portrait,Night Glows,Grand Finale,Cities of the Past,ImaginaryLandscapes. His mural paintingBlack and Whiteis displayed in the lobby of the CCP's BulwagangCarlos V. Francisco (Little Theater). His sculpture of a stainless steel cube is located in front of theBenguet Mining Corporation Building in Pasig.

14. Mang Ernesto works as a candle maker. He has to produce more than 200 pieces of candles to earn ₱2 per candle. If the number of candles produced is from 100 to 200, he will earn ₱1.50 per candle. He will then get ₱1 per candle for the number of candles produced less than 100.


Answer:

(See attached image)

f(x) =   { 2x     if x > 200 }

          { 1.5x   if 100 ≤ x ≤ 200 }

          { x         if x < 100 }

Step-by-step explanation:

(Assuming that you want to create a representation for the problem.)

It is important to remember the different “inequality symbols ( >, <, ≥, ≤ )"Next, our representation is known as the “Piecewise function” - a function that is made up of two or more formulas. Every formula has its own domain, and the domain is the set of all possible values for which the function is defined.How to write it?

                  [tex]f(x)=\left \{ {{formula \ 1 \ if \ x \ is \ in \ domain \ 1} \atop {formula \ 2 \ if \ x \ is \ in \ domain \ 2}} \right.[/tex]

In our case, we have three formulas and domains.

...

First, identify all the conditions.

1st condition: If he produces more than 200 pieces of candles, then each candle is worth ₱2.002nd condition: If he produces greater than or equal to 100 but less than or equal to 200 candles, then each candle is worth ₱1.503rd condition: If he produces less than 100 candles, then each candle is worth ₱1.00

Second, let x be the number of candles. Here, we also use inequality symbols.

Therefore:

1st condition:    if x > 200, then x is multiplied by 2 (php)2nd condition:  if 100 ≤ x ≤ 200, then x is multiplied by 1.50 (php)1st condition:    if x < 100, then x is multiplied by 1 (php)

Finally, simplify using the Piecewise Function:

   f(x) =    { 2x     if x > 200 }

               { 1.5x   if 100 ≤ x ≤ 200 }

               { x         if x < 100 }

...

Say that he produces 205 candles, and 205 is greater than 200.

Then use f(x) = 2x for x > 200

                f(205) = 2 (205)

                f(205) = 410

He’ll earn ₱410 for 205 candles.

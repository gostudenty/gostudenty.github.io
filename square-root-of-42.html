6. 4. Which number has a square root between 6 and 7?A. 32B. 36C. 42D. 49​


Answer:

C. 42

Step-by-step explanation:

Let the number N be the number whose square root is between 6 and 7.

If 6 and 7 are roots of perfect square, then N is the perfect square of the number between them.

To find N, square both given numbers:

6² = (6) × (6) = 36

7² = (7) × (7) = 49

Therefore:

6² < N < 7²  36 < N < 49

A.   32 (not true, because it's less than 36)

B.  36 (not true, because it's the square of given number 6)

C.  42 (TRUE, because it's BETWEEN 6² and 7²)

D.  49 (not true, because it's the square of the given number 7)

ANSWER:   36 < 42 < 49
